## Overview

Bet Analyzer is a web app for exploring e-sports betting angles by combining matchup information, odds comparison, and probability-oriented analysis. It is built to let users evaluate several options quickly rather than checking one market at a time or comparing sources manually.

The project is centered on current or real match data and blends statistical evaluation with Panda API integration. It supports well-known e-sports titles including League of Legends, Counter-Strike 2, and Valorant, so it fits workflows that need multiple game comparisons inside a single browser-based interface.
